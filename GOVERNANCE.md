# Governance

Tiago Silva is the current project maintainer and release owner. The maintainer
triages issues, reviews contributions, decides release timing, and has final
responsibility for the supported public contract.

Routine fixes may be accepted through pull requests. Changes to scope, public
API compatibility, Gymnasium semantics, emulator determinism, packaged states,
or performance acceptance rules should be discussed in an issue before code is
written. Decisions should prefer reproducible evidence and the durable contract
in `SPECS.md`.

Additional maintainers may be added after sustained, trusted participation.
Governance changes will be recorded in this file.
