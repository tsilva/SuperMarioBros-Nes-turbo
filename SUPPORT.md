# Support

Use [GitHub Discussions](https://github.com/tsilva/SuperMarioBros-Nes-turbo/discussions)
for usage questions and GitHub Issues for reproducible bugs or concrete feature
requests. Security issues must follow [SECURITY.md](SECURITY.md).

When reporting a bug, include the package version, Python version, operating
system and architecture, constructor options, complete traceback, and a minimal
reproduction. Report only the ROM SHA-256; never upload or attach a ROM.

The maintained scope is the supported Super Mario Bros NES mapper 0/NROM ROM,
the documented Gymnasium vector API, and the platforms listed in `README.md`.
The project cannot help users acquire game content and does not provide a
general NES compatibility service or individualized reinforcement-learning
consulting.
